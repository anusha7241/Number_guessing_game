# Number Guessing Game 
A simple Python game where the player tries to guess a randomly generated number between 1 and 100.

# Description

This project is a beginner-friendly Python application that helps users practice logic building.
The computer generates a random number, and the user must guess it with hints provided after each attempt.

# Features

* Random number generation using Python
* User input handling
* Hint system (Too High 📈 / Too Low 📉)
* Tracks number of attempts
* Simple and interactive gameplay

# Technologies Used

* Python
* Built-in `random` module

# How to Run

1. Make sure Python is installed on your system
2. Download or clone this repository
3. Open terminal/command prompt in the project folder
4. Run the following command:
5. 
python guess_game.py
